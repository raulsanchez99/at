p span {color:red} se aplica el color a todos los descendientes de span
---------------------

1. p a span em { text-decoration: underline; }

Esto significa que cualquier <em> dentro de un <span>, que a su vez esté dentro de un <a>, que esté dentro de un <p> tendrá subrayado (text-decoration: underline).
Ejemplo afectado:
<p>
  <a href="#">
    <span>
      <em>Texto subrayado</em>
    </span>
  </a>
</p>

2.p, a, span, em { text-decoration: underline; }

Esto aplica subrayado a todos los elementos <p>, <a>, <span> y <em>, sin importar su posición en el HTML.
Esto significa que el subrayado estará presente en cualquier lugar donde estos elementos aparezcan.

3.p a { color: red; }

Esta regla hace que todos los enlaces (<a>) dentro de un <p> tengan color rojo.
Ejemplo afectado:

<p>
  <a href="#">Este enlace será rojo</a>
</p>

4.p * a { color: red; }

Aquí p * a significa cualquier <a> dentro de un <p>, sin importar cuántos niveles de elementos haya entre ellos.
Es decir, el enlace puede estar dentro de un <div>, un <span>, etc., y aún así se aplicará la regla.
Ejemplo afectado:
<p>
  <span>
    <div>
      <a href="#">Este enlace también será rojo</a>
    </div>
  </span>
</p>
